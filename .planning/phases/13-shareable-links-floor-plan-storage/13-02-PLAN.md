    **Data fetching flow:**
    1. Extract `token` from `params` (Next.js 16: `params` is a Promise): `const { token } = await params;`
    2. Create anon Supabase client via `createClient()` from `@/lib/supabase/server`
    3. Call `getShareLink(token, supabase)` — uses the `get_share_link` security-definer RPC
    4. If `getShareLink` returns null: call `checkShareToken(token, supabase)` to distinguish expired vs not-found
       - If `token_exists === true && token_expired === true`: render expired page (D-12)
       - Otherwise: call `notFound()` from `next/navigation` (D-13 handled by Next.js 404 page)
    5. If link found: fetch estimate using **service-role client** (critical — anon client blocked by RLS):
       ```typescript
       import { createServiceClient } from "@/lib/supabase/service";
       const serviceClient = createServiceClient();
       const { data: estimateData } = await serviceClient
         .from("estimates")
         .select("id, version, label, project_inputs, result, created_at")
         .eq("id", link.estimate_id)
         .single();
       ```
       Do NOT use `getEstimate()` directly — it creates an anon client. Use the service client inline.
    6. If estimate not found: call `notFound()`

    - Nelo logo (same SVG as topbar) + "NELO" text
